## Core Findings

### Mise-en-Scene: Everything in the Frame Is a Statement

French for "placing on stage." In cinema, it means everything the camera sees — not just what happens, but how the world is arranged to encode meaning before anyone speaks. [VERIFIED — Wikipedia, StudioBinder, NFI]

**The Six Elements:**

| Element | What It Is | What It Encodes |
|---------|-----------|-----------------|
| Setting / Set Design | Physical environment | Time period, character's inner state, power, decay or growth |
| Lighting | Quality, direction, color of light | Safety/danger, clarity/confusion, hope/despair |
| Costume and Makeup | What subjects wear, how they look | Class, transformation, belonging or alienation |
| Actor Position and Movement | Where subjects stand, how they move | Power dynamics, emotional state, relationship to environment |
| Props | Objects in the scene | Character history, symbolic meaning, foreshadowing |
| Composition | How all above are arranged in frame | Focus of attention, narrative priority, spatial relationships |

**The core principle:** Every element is a deliberate statement. A cluttered desk says "overwhelmed" without anyone saying it. A character in a doorway between two rooms encodes "between two worlds" without a word. [VERIFIED]

**Operational rule:** Before placing any element in a frame (or on a web page), ask: what does this element say about the subject? If it says nothing, it is noise — remove it. If it says the wrong thing, it is disinformation — replace it. [VERIFIED]

### Visual Metaphor: Making Abstract Concepts Visible

A visual representation of an abstract concept through an image suggesting an association. Not decoration — a load-bearing structural element carrying meaning the viewer feels before they can articulate it. [VERIFIED — StudioBinder, filmlifestyle.com]

**Mechanism: substitution.** Replace the abstract concept with a concrete visual sharing its essential property. [VERIFIED]

| Abstract Concept | Visual Metaphor | Shared Property |
|-----------------|----------------|-----------------|
| Hope in confinement | A rock hammer (Shawshank) | Small but persistent, breaks through |
| Grief holding you back | A floating house (Up) | Literally moves with him, anchors |
| Inner conflict | Mirrors, reflections | The self split against itself |
| Confusion / overwhelm | Noise, static, chaos | Direct sensory equivalent |
| Clarity / insight | Single beam of light | Cuts through, isolates the essential |

**Techniques for building visual metaphors:** Object substitution, character-as-concept (Inside Out: Joy IS a character), environmental-as-concept (cluttered room = inner turmoil), color-as-concept (warm = safe, cool = isolated), scale manipulation (important thing large regardless of real-world scale), juxtaposition (place metaphor next to its opposite). [VERIFIED]

### Show Don't Tell: Specific Techniques

Cinema and web are visual platforms. Images and composition carry the story. Text confirms what the eye already knows. If you need text to explain what you mean, the visual has failed. [VERIFIED — Wikipedia, NoFilmSchool]

**The test:** Write down the statement you want to make. Ask: can I remove the text and make a viewer feel the same thing through visuals alone? If no, the visual is not working yet. [VERIFIED]

**Techniques:**

1. **Action over exposition:** Don't tell "she is exhausted." Show: she drops her bag without looking, sits at the edge of the chair, stares at nothing. [VERIFIED]

2. **Environment as character state:** Don't tell "he's lost control." Show: every surface covered, path through the room is narrow, one area is clean — that tells you what he still cares about. [VERIFIED]

3. **Before-and-after without narration:** Two frames, same location, different light, different posture. Viewer resolves the gap. [VERIFIED]

4. **Micro-behavior as macro-signal:** Small physical details encode large emotional states. A character who straightens objects when stressed tells you something they would never say. [VERIFIED]

5. **What's absent:** An empty chair at a dinner table tells a story. A photo turned face-down. Negative space encodes meaning. [VERIFIED]

**Web translation:**

| Film Technique | Web Equivalent |
|---------------|----------------|
| Actor behavior under pressure | UI state changes (hover, loading, error) |
| Environment as character state | Page density: cluttered = complex, sparse = clear |
| Before-and-after without narration | Scroll-revealed transformation |
| Micro-behavior | Micro-interactions: button feedback, number animation, pulse |
| What's absent | Negative space on results page — silence around the number = importance |

### Color as Narrative Device

Color is not decoration or mood. It is a narrative layer encoding the emotional arc across time. The shift FROM one state TO another IS a plot event. [VERIFIED — TED Ideas, fourthwallcontent.com, Academy of Animated Art]

**Color arc vs. color mood:** A color mood is static ("this scene is warm"). A color arc is dynamic ("the film starts warm and bleeds cold as the character loses everything"). The change encodes the arc. [VERIFIED]

**Color script:** Filmmakers create a map of every scene's dominant hue across the entire film before shooting. This lets them see the arc as color. For web/video: map the color state of each section or animation beat before building. The color transition IS part of the story. [VERIFIED — Academy of Animated Art]

### Composition as Storytelling

Composition determines narrative role — who has power, what has attention, what is central vs. marginal. [VERIFIED — StudioBinder, Beverly Boy Productions]

**Rule of thirds:** Frame divided 3x3. Intersection points = natural eye travel. Center frame = authority, power (Kubrick). Off-center = dynamic, in motion. Edge of frame = isolation, vulnerability. [VERIFIED]

**Leading lines:** Lines that create visual vectors — roads, architecture, shadows, gaze direction. Guide where the viewer looks next. Diagonal = energy, instability. Horizontal = rest, stability. Vertical = power, aspiration. Curved = softness, organic flow. [VERIFIED]

**Lead room:** If a subject looks or moves in a direction, leave space in that direction. Space ahead = possibility. No space = confinement. [VERIFIED]

**Framing within the frame:** Doorways, windows, arches frame the subject. Draws eye, contextualizes, encodes "this subject is observed, contained, or central." [VERIFIED]

**Negative space:** Empty space around a subject elevates it. A number alone on a dark background with space is louder than a number surrounded by supporting text. [VERIFIED]

**Web translation:**

| Film Composition | Web Equivalent |
|-----------------|---------------|
| Center frame = authority | Hero element centered, full-width |
| Off-center = dynamic | Cards at intersection points, asymmetric layouts |
| Edge = marginalized | Secondary content at edges (nav, footnotes) |
| Lead room = possibility | Generous padding after CTA |
| Frame within frame | Calculator inside card, card inside dark section |
| Negative space = importance | Result number alone, minimal surrounding |
| Leading lines | Dividers, borders, whitespace vectors guiding scroll |
| Convergence point | The result — everything points toward it |

### Environmental Storytelling

The art of arranging space around a subject so it implies history, situation, and context without stating any of it. [VERIFIED — Game Developer, IntechOpen, TMFF]

**Core technique: show the aftermath, not the event.** Instead of showing a fight, show overturned furniture, broken glass. The viewer reconstructs the cause. Creates more engagement (viewer is active), more emotional impact (imagination exceeds what can be shown), more efficiency (one frame of aftermath replaces minutes). [VERIFIED]

**Techniques:**
1. **Trace evidence** — something happened here; show the physical trace
2. **Contextual objects** — objects implying a specific kind of person or life
3. **Spatial layout as power map** — who has big space, who has small
4. **Contrast between zones** — one area clean, one area chaos
5. **Light and shadow in environment** — where light falls encodes safety/danger
6. **Architecture as attitude** — open vs. closed, high ceilings vs. low, hard edges vs. soft

**Web translation:** The "environment" is the page structure around the calculator, the color of the section, the typography density, the spacing. If the result section has expansive negative space, the environment says "this answer matters." If surrounded by dense competing elements, it says "one of many things." The environment is not decoration — it is part of the statement. [VERIFIED]

### Storyboarding and Beat Boards

A storyboard is a graphic organizer of still images in sequence to pre-visualize a motion picture. The tool that reveals problems before they are expensive to fix. [VERIFIED — Wikipedia, StudioBinder]

**Beat boards:** One frame per major emotional beat. Tests whether the story has the right emotional arc. Not every shot — just the moments that must land. [VERIFIED]

**The director's question sequence for each beat:**
1. What does the viewer see?
2. What do they feel?
3. What do they understand (consciously or intuitively)?
4. What do they want to happen next?
5. How do we transition to satisfy the previous setup and create new tension? [VERIFIED]

### Web as Directing Medium

Layout IS composition. Scroll IS camera movement. Animation IS mise-en-scene. These are not metaphors — they are the same tools applied to a different medium. [VERIFIED — UX/UI sources]

**Layout = Film Frame:**

| Directing Tool | Web Equivalent |
|---------------|---------------|
| Camera frame / shot | Viewport (visible before scroll) |
| Cutting between shots | Section breaks, transitions |
| Close-up | Large typography, zoomed single element |
| Wide shot | Full-width, small text, multiple elements |
| Extreme close-up | Single number, isolated, full-width |

**Scroll = Camera Movement:**

| Camera Movement | Scroll Equivalent |
|----------------|------------------|
| Dolly in | Element scales up on scroll approach |
| Dolly out | Zoomed element reveals full composition |
| Pan | Horizontal scroll, elements entering from sides |
| Crane shot | Long scroll from overview to detail |
| Rack focus | Blur one element, focus another on scroll trigger |
| Parallax | Background and foreground at different scroll speeds |

**Animation Properties as Meaning:**

| Property | What It Encodes |
|----------|----------------|
| Ease-in (slow start) | Natural, things have weight |
| Ease-out (fast start, slow finish) | Coming to rest, arrival |
| Linear | Mechanical, artificial |
| Long duration (800ms+) | Deliberate, weighted, important |
| Short duration (150-200ms) | Snappy, responsive |
| Sequential (stagger) | Reading sequence, hierarchy |
| Simultaneous | Arrival, completeness |

**Scrollytelling:** Scroll position as timeline. New information reveals, old transforms, context builds. This is montage — the director controls sequence and timing of information release. Each scroll-reveal should answer one question the previous beat raised. [VERIFIED — UI Deploy, Vev Design]

### The Director's Toolkit: Seven Questions Before Any Sequence

1. What emotion does the viewer feel at the START?
2. What emotion do we want at the END? (What is the arc?)
3. What is the ONE visual metaphor encoding the arc?
4. What is the color state at start and end?
5. Where is the viewer's eye at the START of each beat?
6. What changes in the environment to signal the arc?
7. What does the viewer do to advance? (Scroll, click, wait — all are choices putting them in a role.) [VERIFIED]

### Three Levels of Visual Storytelling Mastery

**Level 1 (Competent):** Every element has been considered. Nothing accidental. "What does this say?" can be answered for every element. [VERIFIED]

**Level 2 (Good):** Elements form a coherent system. Color, composition, movement, environment all tell the same story. No contradictions. [VERIFIED]

**Level 3 (Director-level):** The sequence has an emotional arc. The viewer feels something different at the end than at the start, and the visual system — not the text — created that change. [VERIFIED]

### Lighting as Environmental Storytelling

From the lighting design source: every lighting choice is a mise-en-scene decision. [VERIFIED]

**Chiaroscuro in web:** Near-black background, single illuminated focal point, everything else fades. The technique is not about showing everything — it is about selecting what to reveal. The Netflix homepage is cinematic chiaroscuro: dark background, hero image pops, everything else recedes. [VERIFIED]

**Motivated lighting in UI:** When the result card itself is the light source, it creates a world with rules. The glow is motivated — the card emits light, illuminating surroundings. Unmotivated glow (random background blobs with no implied source) reads as decorative noise. [VERIFIED]

## Operational Rules

1. **Before placing any element on screen, ask: what does this element say about the subject?** If the answer is "nothing," remove it (noise). If it says the wrong thing, replace it (disinformation). This applies to every web element as it applies to every film frame. [VERIFIED]

2. **When staging a result reveal, animate the result first (or exclusively) while the rest of the UI holds still.** The most important element enters last. Do not animate multiple elements simultaneously if one is more important. [VERIFIED — Disney staging principle]

3. **When building visual metaphors, use substitution: replace the abstract with a concrete visual sharing its essential property.** The metaphor must carry meaning the viewer feels before they articulate it. If it requires explanation, it has failed. [VERIFIED]

4. **When composing a result page, use negative space as the primary emphasis tool.** A number standing alone with space around it is louder than a number surrounded by supporting text. The convergence point of all leading lines should be the result. [VERIFIED]

5. **When planning any multi-beat sequence, write the seven questions first.** Start emotion, end emotion, metaphor, color state, eye position, environment change, user action. If any are unanswered, the sequence is not ready to build. [VERIFIED]

6. **When the environment (page structure, spacing, density) around a result changes, that IS storytelling.** Expansive negative space says "this answer matters." Dense competing elements say "one of many things." Design the environment, not just the element. [VERIFIED]

7. **When translating scroll to camera movement, match the movement type to the narrative intent.** Element scaling up on scroll = dolly in (intimacy). Zoomed element revealing full composition = dolly out (context). Horizontal scroll = pan (survey). Blur shift on scroll = rack focus (attention transfer). [VERIFIED]

## Open Questions

- Film visual storytelling assumes sustained attention (dark theater, no phone). Web users context-switch constantly. Techniques calibrated for sustained attention may not transfer at assumed intensity.
- Web users scan in F-patterns and Z-patterns. They may not register environmental details outside the scan path. Environmental storytelling on web is less studied than in games or film.
- Film audiences look where the director points. Web users scan in patterns partially independent of the director's intent. Composition works with the scan pattern or against it — knowing the scan pattern matters.
- Film is passive; web is interactive. The best web directing uses user action (scroll, hover, tap) as part of the story. Pure film translation ignores this layer.
- Whether "show don't tell" applies fully to health data. Health data may require verbal framing for trust. If users test better with text explanations than visual demonstrations, the blanket application fails.
